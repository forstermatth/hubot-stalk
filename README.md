# hubot-stalk

Hubot knows what your team is doing.

See [`src/stalk.coffee`](src/stalk.coffee) for full documentation.

## Installation

In hubot project repo, run:

`npm install hubot-stalk --save`

Then add **hubot-stalk** to your `external-scripts.json`:

```json
["hubot-stalk"]
```

## Sample Interaction

```
user1>> hubot hello
hubot>> hello!
```
